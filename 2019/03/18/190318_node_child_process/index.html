Nodejs源码之 child_process

发表于 2019-03-18 | 分类于 SourceCode | 浏览 次

　 在 Node.js 中，只有一个主线程执行所有操作，当执行某个CPU密集型任务时，后续操作都需要等待，此时可以通过 child_process 模块开启多个子进程来通信、协作、执行其他任务，多个子进程间可以共享内存空间，也可以通过子进程的互相通信来实现信息的交换。熟悉 shell 的同学，也可以使用它来做很多有意思的事情或工具，比如批量替换、增量部署等等。

child_process模块

常用核心 API

创建异步进程

  • spawn(command[, args][, options])
  • execFile(file[, args][, options][, callback])
  • exec(command[, options][, callback])
  • fork(modulePath[, args][, options])

创建同步进程

  • spawnSync(command[, args][, options])
  • execFileSync(file[, args][, options])
  • execSync(command[, options])
